# MediTrack - Clinic & Appointment Management System

## Overview
MediTrack is a comprehensive clinic management system built in Java that manages patients, doctors, appointments, and billing.

## Features
- 👨‍⚕️ Patient registration and management
- 👩‍⚕️ Doctor management with specializations
- 📅 Appointment booking and tracking
- 💰 Billing with tax calculations
- 📊 Analytics and reports
- 💾 CSV data persistence

## Prerequisites
- Java JDK 17 or higher
- Git (optional)
